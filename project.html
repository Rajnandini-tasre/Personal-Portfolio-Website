Employee Management System – Spring MVC

The Employee Management System is a web application built using Java, Spring MVC, Hibernate, MySQL, HTML, CSS, and Bootstrap. It enables organizations to manage employee data efficiently with CRUD functionalities.

  • Add, update, delete, and view employees.
  • Search employees by name or department.
  • Responsive design using Bootstrap.

Technologies: Java, Spring MVC, Hibernate, MySQL, HTML, CSS, Bootstrap

Student Result Management System – Spring Boot
